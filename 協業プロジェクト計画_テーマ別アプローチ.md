# ヤマエ久野 物流改革プロジェクト
## 協業による段階的ソリューション実現計画

---

## 1. プロジェクトの基本方針

### 協業アプローチの特徴

```
【特徴】
✓ テーマ別プロジェクトで段階的に成果創出
✓ 現場実務に深く入り込み、実態を理解
✓ クイックウィンで早期効果を実証
✓ パイロット→検証→横展開の確実な進め方
✓ 貴社メンバーと協働し、ノウハウ移転
```

### プロジェクト目標

| 目標 | 内容 |
|------|------|
| **短期目標（6ヶ月）** | 現状可視化、クイックウィン創出、効果実証 |
| **中期目標（18ヶ月）** | コアシステム導入、主要拠点展開、年間効果50%達成 |
| **長期目標（36ヶ月）** | 全社展開完了、目標効果100%達成、自走体制確立 |

## 2. 10の重点テーマ

### テーマ1：在庫可視化・実態把握プロジェクト

**目的：** 全拠点・全SKUの在庫実態を正確に把握し、問題在庫を特定

**活動内容：**
- 全拠点の在庫データ収集・統合
- SKU別在庫分析（ABC分析、滞留分析）
- 拠点間在庫偏在の可視化
- 不動在庫・長期滞留在庫のリストアップ
- 在庫精度検証（実地棚卸との突合）

**成果物：**
- 在庫実態レポート
- 滞留在庫リスト（金額・期間別）
- 在庫偏在マップ
- 在庫削減ターゲットリスト

**期待効果：**
- 在庫実態の正確な把握
- 即座に削減可能な在庫の特定（200-500百万円）
- 在庫管理体制の問題点明確化

**期間：** 2-3ヶ月

---

### テーマ2：需要予測精度向上プロジェクト

**目的：** AI/統計手法による需要予測モデルを構築し、発注精度を向上

**活動内容：**
- 過去3年分の販売データ分析
- 季節変動・トレンド・特異日の分析
- 予測モデル開発（統計モデル＋機械学習）
- パイロット商品群での予測精度検証
- 発注ルール見直し

**成果物：**
- 需要予測モデル（カテゴリー別）
- 予測精度レポート
- 新発注ルール案
- パイロット検証結果

**期待効果：**
- 予測精度向上：誤差率±15% → ±5-8%
- 欠品率削減：50%削減
- 過剰在庫削減：300-600百万円

**期間：** 3-4ヶ月

---

### テーマ3：適正在庫基準策定プロジェクト

**目的：** 科学的手法で適正在庫基準を策定し、在庫削減と欠品防止を両立

**活動内容：**
- SKU別需要変動分析
- ABC分類（重要度×変動性）
- 安全在庫・発注点計算
- リードタイム分析
- サービスレベル設定
- 適正在庫基準の策定

**成果物：**
- SKU別適正在庫基準表
- ABC分類マスタ
- 在庫方針書
- 削減ロードマップ

**期待効果：**
- 適正在庫の明確化
- 在庫削減：800-1,500百万円
- 在庫回転日数：7-10日短縮

**期間：** 2-3ヶ月

---

### テーマ4：配送ルート最適化プロジェクト

**目的：** 配送ルート・配車計画を最適化し、配送コストを削減

**活動内容：**
- 配送実態調査（1-2週間の詳細データ収集）
- 配送先・配送量・配送時間の分析
- 積載効率・実車率の測定
- ルート最適化シミュレーション
- 配車計画ルールの見直し
- パイロット実施（1-2ルート）

**成果物：**
- 配送実態分析レポート
- 最適ルート案
- 配車計画ルール
- 削減効果試算

**期待効果：**
- 配送距離削減：10-15%
- 積載効率向上：65-70% → 80-85%
- 配送コスト削減：500-800百万円/年

**期間：** 3-4ヶ月

---

### テーマ5：倉庫オペレーション改善プロジェクト

**目的：** 倉庫作業の効率化・標準化により生産性を向上

**活動内容：**
- 作業実態調査（タイムスタディ、作業観察）
- ムダ・ムリ・ムラの洗い出し
- レイアウト分析・最適化
- ピッキング動線の改善
- 作業標準の策定
- 改善施策の実施・効果測定

**成果物：**
- 作業分析レポート
- レイアウト改善案
- 作業標準書
- 改善効果測定結果

**期待効果：**
- ピッキング生産性：20-30%向上
- 作業時間削減：15-25%
- 人件費削減：50-100百万円/年

**期間：** 3-4ヶ月（拠点あたり）

---

### テーマ6：物流コスト可視化プロジェクト

**成果物：**

- データ連携仕様書
- VMI運用マニュアル
- 効果検証レポート

**期待効果：**
- 在庫削減：パイロット対象で20-30%削減
- 欠品率削減：50%削減
- 発注業務効率化

**期間：** 6-9ヶ月

**期待効果：**
